# Simple Interest Calculator

This is a final project for the IBM course **"Introduction to Web Development with HTML, CSS, and JavaScript"** on Coursera.

## Project Description
The Simple Interest Calculator is a web-based tool that allows users to calculate the interest earned on a principal amount over a specific period at a given interest rate.

### Features
- Input field for the principal amount.
- Range slider for selecting the interest rate.
- Dropdown menu for selecting the number of years.
- "Compute Interest" button to perform the calculation.
- Validation to ensure the principal amount is a positive number.
- Dynamic result display with highlighted values.

## How to Run
1. Clone this repository.
2. Open `index.html` in any web browser.

# حاسبة الفائدة البسيطة

هذا هو المشروع النهائي لدورة IBM **"مقدمة في تطوير الويب باستخدام HTML و CSS و JavaScript"** على منصة كورسيرا.

## وصف المشروع
حاسبة الفائدة البسيطة هي أداة ويب تتيح للمستخدمين حساب الفائدة المكتسبة على مبلغ أصلي خلال فترة زمنية محددة وبسعر فائدة معين.

### المميزات
- حقل إدخال للمبلغ الأصلي.
- شريط منزلق لاختيار سعر الفائدة.
- قائمة منسدلة لاختيار عدد السنوات.
- زر "حساب الفائدة" لإجراء العملية الحسابية.
- التحقق من صحة البيانات لضمان أن المبلغ الأصلي رقم موجب.
- عرض ديناميكي للنتائج مع تمييز القيم المحسوبة.

## كيفية التشغيل
1. قم بنسخ المستودع.
2. افتح ملف `index.html` في أي متصفح ويب.
